# ModelSynk - a simple way to sync models
ModelSynk is a tool that is designed to do one thing and that one thing well. Sync any model definition from any source to any language that we have defined. This is currently an in progress application and will start with C# and TypeScript models that can be generated from SQLite, MS-SQL, PostgreSQL, and MySQL/MariaDB in that order.
